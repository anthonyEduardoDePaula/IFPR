HTML

A linguagem HTML - Hypertext Markup Language é uma linguagem de marcação de conteúdo. Tem por objetivo definir a estrutura do conteúdo como por exemplo: definir parágrafos, títulos, listas, tabelas, imagens, audio, videos links entre outros.

Tags de Títulos

As tags de títulos são definidas pela letra h e possui seis níveis de títulos. Exemplo:

Título nível 1

Título nível 2

Título nível 3

Título nível 4

Título nível 5
Título nível 6

Tags de destaque de texto

Para realizar o destaque de um texto podemos usar o negrito ou itálico.

O negrito é definido pelas tags "b" ou "strong". Exemplo:

Texto negritado

Texto negritado com "strong"

O itálico é dado pelas tags "i" ou "em". Exemplo:

Texto em itálico

Texto em itálico com "em"
